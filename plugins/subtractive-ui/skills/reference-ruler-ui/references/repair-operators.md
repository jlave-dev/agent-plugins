# Repair Operators

Run one operator, re-capture, then accept or reject.

- Row density: adjust row height, padding, rule weight, control size; guard viewport budget.
- Lane centerline: center marker, text, trailing value, chevron, and rail in the measured lane.
- Typography: import/load better font candidates; tune role size, weight, line height, and crop.
- Micro control anatomy: use precise CSS/SVG for small controls, rings, markers, badges, and tabs.
- Parent before child chart/table: repair shell, plot/card box, and section bands before marks/rows.
- Section-stack plus chart type: match dashboard y-rulers and the actual chart family together.
- Table-card parent-to-child: fix card header, table-head, rules, insets, then icon/logo boxes.
- Repeated-card queue rhythm: tune rail cadence, card budget, action state, title wrap, visible count.
- Title-token line: extract title tokens before shrinking text mixed with metadata.
- Icon metaphor/silhouette: change icon family/custom SVG, size, stroke, slot, and alignment.
- Logo/brand slot: improve mark silhouette without moving the wordmark.
- Editor badge-anchor: align nodes to badge/port centers and update connectors together.
- Inspector field-column: tune label column, field x/width/height, row interval, chevrons, code box.
- Sidebar row-label: tune columns, gaps, type, nowrap, active state, trailing icon.
- Canvas texture: tune dot/grid pitch and contrast from clean empty zones.
- Command-bar cadence: tune group x/width, icon count, gaps, action anchors.
- Node icon-tone anatomy: tune typed icon tile tone/silhouette; reject OCR labels unless crop improves.
- Template-library icon taxonomy: use distinct category icons; guard row/list crop and graph-data boundary.
