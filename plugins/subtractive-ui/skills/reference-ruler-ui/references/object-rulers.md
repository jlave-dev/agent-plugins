# Object Rulers

Measure the reference object, then measure the rendered object with the same ruler.

- Universal: box, gap, row interval, centerline, baseline, stroke, darkness, opacity, radius, type metrics, color/state, parent slot, sibling anchor.
- Rows/lists/tables: lane center, separator y/darkness, text/trailing centers, visible count. For tables, fix card header and table-head bands before row height.
- Mobile: status/top bars, active section, fixed nav, home indicator, above-fold budget.
- Icons/marks: metaphor, silhouette, slot, stroke, accent area, bbox, centroid. For logo rows, preserve wordmark x.
- Editors/canvases: coordinate system, badge/port centers, connector endpoints, selected outline, dot-grid pitch/contrast.
- Forms/inspectors: label column, field x/width/height, row interval, chevrons, code-box bounds.
- Dashboards/charts: shell columns, section-stack y rulers, card bands, plot box, gutters, axes, legends, labels, gridline rhythm.
- Variable data: mask arbitrary traces, sparklines, graph particulars, generated records, and live values unless the task fixes that state.
